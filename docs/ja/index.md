## バイリンガル規約

このプロジェクトでは、ドキュメントを2つの言語で管理しています：

- **英語** (`docs/en/`) — 原文（Source of Truth）。すべての変更はここから始まります。

Claude Code はコンテキストウィンドウの使用量を最小限にするため、英語のドキュメントのみを読み込みます。人間のコントリビューターはどちらの言語でも参照できます。
