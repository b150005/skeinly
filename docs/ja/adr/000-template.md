# ADR-000: アーキテクチャ決定記録テンプレート

## ステータス

承認済み

## コンテキスト

アーキテクチャ決定記録（ADR）は、重要なアーキテクチャ上の決定をそのコンテキストと結果とともに記録します。このテンプレートは、これらの決定を記録するための一貫したフォーマットを提供します。

## 使い方

1. このファイルを `NNN-title.md` としてコピーします（例: `001-use-postgresql.md`）
2. 以下のセクションを自分の決定の詳細に置き換えます
3. ファイルを `docs/en/adr/` と `docs/ja/adr/` の両方に追加します
