アカウント削除

最終更新日: 2026-05-12

いつでも Skeinly アカウントを削除できます。最も早い方法はアプリ内からの削除で、認証済みセッション内で実行されるため待ち時間も本人確認手続きも不要です。ここではアプリにアクセスできない場合の Web ベースのフォールバック手順も説明します。

方法 1 — アプリから削除 (推奨)

  1. スマートフォンで Skeinly を開きます。
  2. 設定 をタップします (ナビゲーションバーの歯車アイコン)。
  3. アカウント セクションまでスクロールします。
  4. アカウントを削除 をタップします。
  5. 確認ダイアログを読み、確認します。

アカウントは即座に削除されます。サインアウトされ、サインイン画面に戻ります。

方法 2 — Web 経由での削除リクエスト

アプリにサインインできない場合 (端末紛失、パスワード忘れ等) のみこちらの方法をご利用ください。Web 経由の削除はオペレーターによる手動本人確認が必要で、最大 7 暦日かかります。

  1. 件名 Account Deletion Request で skeinly.app@gmail.com 宛にメールを送信します。
  2. Skeinly アカウントに登録されているメールアドレスから 送信してください。これが本人確認ステップです — 別のアドレスからの削除リクエストは処理できません (GDPR Article 12(6) に準拠)。
  3. 7 暦日以内に削除完了の確認返信をお送りします。本人確認できなかった場合 (例: From アドレスがどの Skeinly アカウントとも一致しない場合)、確認可能なチャネルからの再送をお願いする返信をお送りします。

削除されるデータ

アカウント削除時に Skeinly のシステムから以下のデータが削除されます:

データ種別削除タイミング
アカウントレコード (メール、サインインメタデータ)即時
プロフィール (表示名、自己紹介、アバター)即時 (カスケード)
パターン、プロジェクト、進捗、編み図、アレンジ即時 (カスケード)
あなたが投稿した提案、コメント、アクティビティ即時 (カスケード)
共有 (送信先・受信先の両方)即時 (カスケード)
プッシュ通知トークン即時 (カスケード)
サブスクリプションレコード (Skeinly 側)即時
アバター画像ファイル (Skeinly Storage)即時
Sentry / PostHog データ (オプトインの場合)匿名 — 匿名 install UUID を提示の上、Sentry / PostHog に直接削除依頼可能
Apple App Store / Google Play 購入履歴Skeinly の管理外 — Apple / Google が管理、各社に直接削除依頼可能
RevenueCat サブスクリプション分析データSkeinly の直接管理外。レコード削除を希望する場合は RevenueCat サポートまで

削除は取り消しできません。 アカウントが削除されると、パターン、プロジェクト、進捗は復元できません。Pro サブスクリプション継続中の場合、削除前に Apple Subscriptions または Google Play から解約しないと請求が続きます (Skeinly アカウントの削除は Apple / Google の IAP サブスクリプションを自動的にキャンセルしません)。

Pro サブスクリプションは個別に解約 (該当する場合)

Skeinly アカウントを削除しても、Apple App Store または Google Play のサブスクリプションは自動的にキャンセルされません。アカウント削除前 (または後) に各ストアで直接解約してください:

  • iOS: apps.apple.com/account/subscriptions または端末の 設定 → Apple ID → サブスクリプション
  • Android: play.google.com/store/account/subscriptions または Play ストア → プロフィール → お支払いと定期購入

気が変わった場合

アカウントが一度削除されると復元できません。同じメールアドレスで再度サインアップは自由ですが、削除済アカウントとのリンクはない完全な新規アカウントとなります。

削除手続きに関するご質問・トラブル? skeinly.app@gmail.com までメールしてください。
