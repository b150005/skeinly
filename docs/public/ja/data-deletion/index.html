データ削除

最終更新日: 2026-05-15

このページでは Skeinly アカウントは保持したまま、あなたが投稿したパターン・プロジェクト・編み図・コメントなどのデータをすべて削除する方法 を説明します。サインイン情報、表示名、アバター、Pro メンバーシップはそのまま保持されます。削除されるのは、あなたが投稿したコンテンツのみです。

アカウント自体を削除したい場合は? アカウント削除 をご利用ください。アカウントそのものを削除します(コンテンツだけでなく)。

削除されるもの / 保持されるもの

削除されるもの

  • あなたが作成したパターン
  • プロジェクトと進捗
  • 編み図、アレンジ、バージョン履歴
  • コメント、提案、提案へのコメント
  • あなたが作成した共有リンク
  • アクティビティ履歴
  • ダウンロード済みシンボルパック(再ダウンロード可能)
  • プッシュ通知トークン(次回サインイン時に自動再登録)
  • あなたが設定したブロック(ブロックリストのみ。相手ユーザーは削除されません)

保持されるもの

  • サインイン情報 (メール+パスワード または OAuth)
  • 表示名と自己紹介
  • アバター画像
  • Pro メンバーシップ (該当する場合)
  • プッシュ通知設定 (自動的に再適用)
  • アカウント作成日

方法 1 — アプリから削除 (推奨)

  1. スマートフォンで Skeinly を開きます。
  2. 設定 をタップします (ナビゲーションバーの歯車アイコン)。
  3. Danger Zone までスクロールします。
  4. すべてのデータを削除 をタップします。
  5. 削除されるもの / 保持されるものの一覧を確認し、続行 をタップします。
  6. 表示されている確認フレーズを正確に入力し、送信します。

データは即座に削除され、パターンライブラリに完了バナーが表示されます。サインインは維持され、Pro メンバーシップも引き続き利用できます。

方法 2 — Web 経由での削除リクエスト

アプリにサインインできない場合 (端末紛失、パスワード忘れ等) のみこちらの方法をご利用ください。Web 経由の削除はオペレーターによる手動本人確認が必要で、最大 7 暦日かかります。

  1. 件名 Data Deletion Request (account-preserved) で skeinly.app@gmail.com 宛にメールを送信します。
  2. Skeinly アカウントに登録されているメールアドレスから 送信してください。これが本人確認ステップです — 別のアドレスからの削除リクエストは処理できません (GDPR Article 12(6) に準拠)。
  3. 7 暦日以内に削除完了の確認返信をお送りします。本人確認できなかった場合 (例: From アドレスがどの Skeinly アカウントとも一致しない場合)、確認可能なチャネルからの再送をお願いする返信をお送りします。

削除は取り消しできません。 パターン・プロジェクト・編み図・コメントが一度削除されると復元できません。この操作は アカウント削除 とは意図的に区別されています — アカウント自体も削除したい場合は、アカウント削除フローをご利用ください。

アカウント削除との違い

Skeinly では関連する 2 つのプライバシーコントロールを提供しています:

  • すべてのデータを削除 (このページ) は、アカウントとPro メンバーシップを維持したまま、あなたが投稿したコンテンツのみを削除します。サインインを失わずに「リセット」したい場合に便利です。
  • アカウント削除 はアカウントレコードと投稿コンテンツを同時に削除します。Skeinly アカウント自体を完全に削除したい場合はこちら。

「すべてのデータを削除」は何度実行しても問題ありません — 冪等な操作です。各実行時点で存在するコンテンツがすべて削除されます。

気が変わった場合

データが一度削除されると復元できません。個別のパターン・プロジェクトを後で復元したい場合は、送信前にキャンセルしてください。削除後はサインインし直して新たに始められます。

削除手続きに関するご質問・トラブル? skeinly.app@gmail.com までメールしてください。
