# ADR-000: Architecture Decision Record Template

## Status

Accepted

## Context

Architecture Decision Records (ADRs) capture important architectural decisions along with their context and consequences. This template provides a consistent format for recording these decisions.

## How to Use

1. Copy this file as `NNN-title.md` (e.g., `001-use-postgresql.md`)
2. Replace the sections below with your decision's details
3. Add the file to both `docs/en/adr/` and `docs/ja/adr/`
